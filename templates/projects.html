SprachlernApp

Die Sprachlern-App ist ein einfaches Python-Programm, das dazu dient, Vokabeln in einer Fremdsprache zu lernen und zu üben. Die App ermöglicht es, Vokabeln zu lernen, Sätze zu lernen, sowie den Lernprozess individuell anzupassen. Vorerst wird nur Deutsch / Englisch unterstützt!

The Sprachlern-App is a simple Python program that is designed to help you learn new words in a foreign language. The app allows you to learn new words, sentences and adjust the learning process individually. Currently only German / English is supported!

LernZwerg - Lern App für Schüler

LernZwerg ist ein Tool, das als Lernhilfe für Schüler entwickelt wurde. Es bietet eine einfache Benutzeroberfläche zum Öffnen, Bearbeiten und Speichern von Textdateien sowie eine Text-to-Speech-Funktion, mit der der Benutzer den Text in der Datei hören kann.

LernZwerg is a tool that was developed as a learning aid for students. It offers a simple user interface to open, edit and save text files as well as a text-to-speech function that allows the user to listen to the text in the file.
